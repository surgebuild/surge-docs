# Native Stable Coins

Rollups can also introduce fragmentation of protocol’s total value locked (TVL) and inherent liquidity. This problem highlights the need for increased interoperability between layer-2 ecosystems.

While Bitcoin rollups are critical for innovation and scalability, their viability hinges on bridges that uphold the same ethos of decentralization and sovereignty.
